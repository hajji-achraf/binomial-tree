Analyse théorique de convergence

Convergence du modèle binomial

Le modèle binomial converge vers la solution de Black-Scholes lorsque le nombre d'étapes tend vers l'infini. Le taux de convergence est généralement en O(1/n) pour les options européennes, où n est le nombre d'étapes.

La vitesse de convergence dépend des paramètres du modèle. Pour le modèle Cox-Ross-Rubinstein (CRR), nous avons:

  • u = eσ√(Δt): facteur de hausse du prix
  • d = e-σ√(Δt): facteur de baisse du prix
  • p = (erΔt - d)/(u - d): probabilité risque-neutre de hausse
Améliorations possibles :
  • Méthodes d'extrapolation de Richardson pour améliorer la convergence
  • Paramétrage de l'arbre pour réduire l'erreur de discrétisation
  • Approches adaptatives avec plus de nœuds dans les régions critiques
